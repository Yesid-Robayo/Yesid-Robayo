# ¡Bienvenido a mi perfil de GitHub! 👋

## 💫 Sobre mí

👨‍💻 Soy **Yesid**, desarrollador de software **Fullstack**. Disfruto construir software de extremo a extremo —de la interfaz de usuario a la lógica del servidor y la base de datos— y, últimamente, también **aplicaciones de escritorio** que interactúan a bajo nivel con el sistema operativo.

🎯 Mi objetivo es crear soluciones que resuelvan problemas reales con una experiencia de usuario cuidada, poniendo el foco en la **calidad, la arquitectura limpia y el detalle**.

📚 Aprendo de forma continua y creo en el poder de la colaboración: comparto lo que sé y busco proyectos que reten los límites de lo que la tecnología puede hacer hoy.
